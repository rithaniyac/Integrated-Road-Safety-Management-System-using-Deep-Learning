An intelligent computer vision-based traffic surveillance system designed to automate the detection of road safety violations using state-of-the-art deep learning techniques. The system integrates **YOLOv12** for object detection, **OpenCV** for real-time video processing, and **Optical Character Recognition (OCR)** for vehicle identification to enable efficient, accurate, and scalable traffic monitoring.

## Overview

The project provides an end-to-end framework for detecting multiple traffic violations, including helmet non-compliance, triple riding, and overspeeding, while automatically recognizing vehicle registration numbers. By combining object detection, vehicle tracking, speed estimation, and license plate recognition, the system minimizes manual intervention and supports intelligent traffic enforcement.

## Key Capabilities

- Real-time vehicle and rider detection
- Helmet compliance monitoring
- Triple riding detection
- Vehicle speed estimation
- Automatic License Plate Recognition (ALPR)
- Intelligent traffic violation identification
- Scalable video-based surveillance

## Technology Stack
